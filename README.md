Автоматизированный тест на базе Selenium с использованием паттерна PageObject и подхода Steps.  
Автоматический тест проверки выбора из 5 предложенных курсов самого популярного, и возможности поступления на этот курс. 

Стек:  
язык : java  
фраймворк : junit5, maven  
библиотеки : selenium, aeonbits, allure  
Требования:  
SDK : java11 и выше  
driver : resources/tests.properties{
chromeDriver.endpoint="скачай chrome-драйвер локально и укажи путь до него в этом поле"
}
Структура проекта:  
src/main/java/pages - PO pattern  
src/main/java/steps -  шаги по автотесту
src/main/java/helpers - входные данные для тестов, скриншотирование, настройка пропертей  
src/test/java/tests - тесты
src/main/resources - пропертии (указывается путь до локального драйвера, а также тестируемый сайт)    
Тестируем:    
[x] Поиск курсов по теме.  
[x] Фильтрация курсов по входным параметрам.  
[x] Выбор самого популярного курса из первых пяти курсов в списке найденных.
[x] Проверка возможности поступления на самый популярный курс.  
Отчёт:
В проекте настроена генерация подробных отчетов Allure. После завершения тестов выполните команду:
'mvn allure:serve'
